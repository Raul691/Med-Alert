# 💊 MedAlert

## 📌 Sobre o Projeto

O MedAlert é um aplicativo desenvolvido em Python utilizando a biblioteca Kivy com o objetivo de auxiliar usuários no gerenciamento de medicamentos.

O sistema permite cadastrar medicamentos, controlar a quantidade disponível em estoque e registrar o consumo diário, ajudando a evitar esquecimentos e a falta de medicamentos essenciais.

O projeto foi desenvolvido como atividade acadêmica da disciplina **Development With Python**.

---

## 🎯 Problema que o Projeto Resolve

Muitas pessoas utilizam medicamentos de uso contínuo e podem enfrentar dificuldades como:

- Esquecimento dos horários de medicação;
- Falta de controle da quantidade disponível;
- Risco de ficar sem medicamentos importantes;
- Dificuldade em organizar tratamentos de longo prazo.

O MedAlert busca oferecer uma solução simples para auxiliar nesse controle.

---

## 🚀 Funcionalidades

### 🏠 Dashboard
- Exibe a quantidade total de medicamentos cadastrados.
- Mostra quantos medicamentos estão com estoque baixo.

### 💊 Cadastro de Medicamentos
- Nome do medicamento.
- Quantidade disponível.
- Horário de utilização.

### 📋 Lista de Medicamentos
- Visualização dos medicamentos cadastrados.
- Exibição das informações registradas.

### 📦 Controle de Estoque
- Registro de consumo através do botão "Tomei".
- Atualização automática da quantidade disponível.
- Alertas para medicamentos com estoque baixo.

### 💾 Persistência de Dados
- Armazenamento em arquivo JSON.
- Recuperação automática dos dados ao iniciar o aplicativo.

## ▶️ Como Executar o Projeto

```bash

```

### 2. Entrar na pasta

```bash
cd MedAlert
```

### 3. Instalar o Kivy

```bash
pip install kivy
```

### 4. Executar o aplicativo

```bash
python main.py
```
